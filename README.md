# User Profile

## Descripción

Este proyecto es una página web de perfil de usuario creada con Next.js y React. Incluye información básica, una lista de intereses y una sección de contacto.

## Instalación

1. Clona este repositorio:
    https://github.com/agustina-sosa00/pruebaTecnicaSaS
2. Instala las dependencias:
   npm install
3. Ejecuta la aplicacion:
    npm run dev
